# Project Instructions — Coffee Brew Lab

## Commit and Push After Every Milestone

After completing each milestone, do the following automatically without waiting for instruction:
1. Stage all changed and new files (`git add`)
2. Commit with a short descriptive message referencing the milestone (e.g. "Milestone 1.3: static pages")
3. Push to the `main` branch on GitHub

Do not skip this step. Do not wait to be asked.
